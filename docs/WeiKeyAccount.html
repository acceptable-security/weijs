    (async) sendTransaction(transaction) → {string}

    This will take the given WeiTransaction, generate the necessary nonce, load the gas price from the RPC if necessary, RLP encode the unsigned transaction, sign that, then store the r/s/v values of the transaction, and then encode that into the RLP signature. After this it will send the transaction using sendRawTransaction.
