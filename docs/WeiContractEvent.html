    WeiContractEvent

    A class that wraps an event for a given contract.

    Constructor

    new WeiContractEvent(wei, abi, address)

    When the WeiContract parses its ABI, it will generate a WeiContractEvent for each event specified in the ABI. When WeiContract#address is called, it will automatically pass the address down to the events it has generated. The WeiContractEvent objects are exposed by the event names directly on the WeiContract objects.
    Source:

    Parameters:
    Name Type Description
    wei Wei The wei instance being used.
    abi Object The ABI of the contract event.
    address string = undefined - Optionally, the address of the contract.
    Examples
    // Listen for any Transfer events emitted from an ERC20 token.
// Which will be checked every 2 seconds
const wei = new Wei(... provider info ...);
const contract = wei.contract(... erc20 abi ...);
contract.Transfer.listen(2 * 1000).on('event', event => {
    console.log(event._owner, "transfered", event._value.toString(), "to", event._spender); 
});
    // Listen for any Transfer events from a specific address from an ERC20 token.
// Which will be checked every 2 seconds
const wei = new Wei(... provider info ...);
const contract = wei.contract(... erc20 abi ...);
contract.Transfer.listen({ _owner: "0x..." }, 2 * 1000).on('event', event => {
    console.log(event._owner, "transfered", event._value.toString(), "to", event._spender); 
});
